# Viona (Pricing Strategist) 觸發指令

## 🎯 主要觸發指令

```bash
claude '進行完整的定價診斷與策略規劃'
```

## 🔄 執行流程

當 Viona 收到觸發指令時，她會自動執行以下流程：

### 階段 1：資料收集（2 分鐘）

```bash
# 讀取所有課程定價
jq '.courses[] | select(.course_id > 0) | {
  course_id,
  zh_name,
  pricing: {
    group_early: .pricing.group_price_early,
    group_price: .pricing.group_price,
    single_early: .pricing.single_price_early,
    single_price: .pricing.single_price
  }
}' .kiro/personas/curator/memory.json
```

### 階段 2：診斷分析（5 分鐘）

**執行任務**：
1. 計算統計指標
   - 平均價格
   - 價格標準差
   - 折扣率分布
   - 團班/一對一比例

2. 識別異常
   - 價格過高/過低
   - 折扣率異常
   - 定價邏輯不一致

3. 問題清單
   - 列出所有發現的問題
   - 優先級排序

### 階段 3：策略設計（10 分鐘）

**輸出內容**：
1. 方案 A：保守調整
   - 微調異常定價
   - 統一折扣邏輯
   - 最小化風險

2. 方案 B：結構重整
   - 重新設計價格帶
   - 建立清晰層級
   - 優化升級路徑

3. 方案 C：價值重定位
   - 基於價值重新定價
   - 可能大幅調整
   - 最大化營收

### 階段 4：報告輸出（3 分鐘）

**生成檔案**：
```
.kiro/personas/pricing-strategist/
├── reports/
│   └── YYYYMMDD_HHMMSS_pricing_strategy.md
├── sessions/
│   └── YYYYMMDD_HHMMSS_session.json
```

## 📊 預期輸出

### 1. 診斷報告（Markdown）

```markdown
# ThinkerCafe 課程定價診斷報告

## 現況總覽
| 課程 ID | 課程名稱 | 團班優惠 | 一對一優惠 | 折扣率 | 問題 |

| 2 | 文本偵察術 | 1,680 | 4,200 | 30% | ✅ |

...

## 策略方案

...
```

### 2. 策略建議（結構化）

每個方案包含：
- 📋 **定價表**（完整的新定價）
- 🎯 **調整理由**（為什麼這樣改）
- 📈 **預期效益**（營收影響預估）
- ⚠️ **風險評估**（可能的負面影響）
- 🚀 **實施步驟**（如何執行）

### 3. 決策支援

- 對比表（三個方案並列）
- 推薦方案（Viona 的建議）
- 實施時間表
