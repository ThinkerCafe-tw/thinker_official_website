# Pricing Strategist (Viona)

## 🎯 角色定位
ThinkerCafe 的定價策略顧問，負責：
- 診斷現有課程定價問題
- 設計可持續的定價體系
- 提供數據驅動的策略建議
- 確保定價支撐商業目標

## 📂 目錄結構
```
pricing-strategist/
├── README.md              # 本文件
├── PERSONA.md            # 完整人格定義
├── TRIGGER.md            # 觸發指令與工作流程
├── sessions/             # 每次分析的完整記錄
├── reports/              # 輸出的策略報告
├── tools/                # 分析工具與腳本
└── templates/            # 報告模板
```

## 🚀 快速啟動

### 步驟 1：切換人格
```bash
.kiro/scripts/switch-persona.sh pricing-strategist
```

### 步驟 2：在新對話中啟動
開啟新的 Claude 對話，輸入：
```
進行完整的定價診斷與策略規劃
```

### Viona 會自動執行
1. 讀取所有課程定價資料
2. 進行統計分析與視覺化
3. 診斷定價問題
4. 提出 2-3 個策略方案
5. 輸出完整報告

## 📊 輸出物

### 1. 診斷報告
- 現況分析（價格分布、異常值）
- 問題識別（定價邏輯混亂）
- 數據視覺化

### 2. 策略方案
- 方案 A/B/C 對比
- 每個方案的預期效益
- 風險評估

### 3. 實施計畫
- 具體定價表
- 過渡期處理
- A/B 測試建議
